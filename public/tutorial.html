FlatBookMark 使用教程

最后更新日期：2026-06-02

快速开始（3 步设置）

第一步：创建书签根文件夹

打开 Chrome 书签管理器，在书签栏中新建一个文件夹，命名为 FlatBookMark（可自定义） 。

第二步：添加行文件夹

在 FlatBookMark 文件夹内创建 2 个子文件夹，分别代表第 1 行和第 2 行。例如： 第一行 、 第二行 。

第三步：添加列文件夹和书签

在每个行文件夹内再创建 5 个子文件夹作为列，然后在每个列文件夹中放入具体书签链接。 以 2×5 布局为例，书签结构为 FlatBookMark → 第一行 → 列1/列2/列3/列4/列5 → 具体书签链接 。

书签结构示意（2×5 布局）：
书签栏 → FlatBookMark → 第一行 → 列1 / 列2 / 列3 / 列4 / 列5 → 各自的具体书签
            → 第二行 → 列1 / 列2 / 列3 / 列4 / 列5 → 各自的具体书签

📖 功能说明

🔀 拖拽排序

直接拖拽书签卡片即可调整顺序，拖拽后的排序会自动保存到 Chrome 书签中。也可以跨区块拖拽，将书签移动到不同的子文件夹。

✏️ 重命名与删除

鼠标悬停在书签卡片上，会出现操作按钮：

  • ✏️ 铅笔图标 — 重命名书签
  • ️ 垃圾桶图标 — 删除书签
  • 🚩 旗帜图标 — 强调书签/取消强调

⭐ 强调书签

点击旗帜图标可将重要书签标记为强调状态。强调的书签会显示红色边框和旗帜标记，方便快速找到关键链接。

🌗 主题切换

点击页面右上角的主题按钮，可在 浅色 和 深色 模式之间切换。深色模式适合夜间使用，减少眼睛疲劳。

📐 布局选择

在设置页可以选择不同的网格布局：

  • 2 × 5 — 两行五列，适合书签较多的场景
  • 2 × 4 — 两行四列，适合宽屏显示器

布局选择后会自动保存到设置中，下次打开新标签页仍然生效。

📂 书签结构（以 2×5 布局为例）

每个网格区块对应一个子文件夹。2×5 布局表示 2 行，每行 5 列 ，书签夹的文件夹结构如下： 

书签栏
└── FlatBookMark（自定义根文件夹名称）
    ├── 第一行
    │   ├── 第一列
    │   │   ├── 书签链接
    │   │   └── 书签链接
    │   ├── 第二列
    │   │   └── 书签链接
    │   ├── 第三列
    │   │   └── 书签链接
    │   ├── 第四列
    │   │   └── 书签链接
    │   └── 第五列
    │       └── 书签链接
    └── 第二行
        ├── 第一列
        │   └── 书签链接
        ├── 第二列
        │   └── 书签链接
        ├── 第三列
        │   └── 书签链接
        ├── 第四列
        │   └── 书签链接
        └── 第五列
            └── 书签链接

根文件夹名称可在设置中自定义，第一行/第二行等子文件夹名称也可自由命名， 它们在页面上会显示为各个分组的标题。

⚙️ 设置页面

点击页面右上角的 ⚙️ 设置 按钮进入设置页面，可以：

  • 修改书签根文件夹名称
  • 切换页面布局（2×5 / 2×4）
  • 切换主题模式（浅色 / 深色）
  • 切换语言（中文 / English）

🔌 弹出窗口与侧边栏

弹出窗口： 点击浏览器工具栏的 FlatBookMark 图标，弹出窗口提供快速入口：

  • 打开书签页 — 在新标签页打开书签网格
  • 使用教程 — 打开本教程页面

侧边栏： 点击 Chrome 侧边栏图标（或右键工具栏图标选择"打开侧边栏"），可以快速访问书签页。

常见问题

打开新标签页显示"未找到书签"怎么办？

请检查：

  1. 书签栏中是否创建了名为 FlatBookMark 的文件夹
  2. 文件夹名称是否与设置页中配置的根文件夹名称一致
  3. 文件夹内是否有子文件夹和书签

如果仍未解决，点击页面右上角的 ️ 设置 重新配置。

拖拽排序后刷新页面又恢复了？

拖拽排序会实时同步到 Chrome 书签。如果刷新后恢复，可能是 Chrome 书签同步延迟，请稍等片刻再刷新。

可以在多台电脑上同步配置吗？

FlatBookMark 的配置数据存储在浏览器本地（ chrome.storage.local ）。如果 Chrome 开启了书签同步，书签数据会自动同步，但扩展的配置（布局、主题）需要在每台电脑上分别设置。

支持 Edge 浏览器吗？

FlatBookMark 目前仅支持 Chrome 浏览器。Edge 浏览器基于 Chromium 内核，可以尝试安装，但部分功能可能不兼容。
