FlatBookMark 隐私政策

最后更新日期：2026-06-03

概述

FlatBookMark 是一款 Chrome 浏览器扩展，用于将您的书签以扁平化网格的形式展示在新标签页上。 我们高度重视您的隐私。本隐私政策旨在清晰、透明地说明本扩展的数据处理方式。

数据收集与使用

FlatBookMark 不收集、不传输、不存储任何用户数据到远程服务器 。 扩展运行时所需的全部数据（书签内容、用户配置、偏好设置）均通过 Chrome 原生 API 实时读取， 或保存在浏览器的本地存储中，不会离开您的设备。

我们不会：

  • 收集或上传您的浏览历史
  • 收集或上传您的书签内容
  • 追踪您的网页访问行为
  • 向任何第三方发送数据
  • 嵌入任何分析脚本、广告代码或追踪器

权限说明

本扩展请求以下 Chrome 权限，每个权限都仅用于实现核心功能：

  • bookmarks — 读取您的书签树结构，以及在您拖拽、重命名、删除书签时同步修改 Chrome 书签。 这是扩展的核心功能，没有此权限无法工作。
  • favicon — 调用 Chrome 内置的 favicon API（ chrome://favicon ），为每个书签显示其网站图标。
  • storage — 使用 chrome.storage.local 在您的浏览器本地存储以下数据：
    • 书签布局配置（根文件夹名称、网格布局）
    • 主题设置（浅色/深色模式）
    • 收藏书签列表
    • 语言偏好

主机权限

本扩展请求 <all_urls> 主机权限，仅用于以下目的：

  • 通过 Chrome 的 favicon API 获取任意网站（无论域名）的图标，以在书签卡片上显示。

扩展不会访问、读取或修改任何网页的内容。此权限仅用于 favicon 图标的跨域获取。

本地存储数据明细

扩展使用 chrome.storage.local 存储以下键值，所有数据仅保存在您的浏览器本地：

  • crx-bookmark-config — 书签根文件夹名称、网格布局（如 2×5、3×3）
  • crx-theme — 当前主题模式（浅色/深色）
  • crx-bookmark-favorites — 被收藏的书签 ID 列表
  • crx-language — 用户选择的语言（中文/English）

第三方服务

本扩展不使用任何第三方追踪服务、分析工具、广告网络或外部 API。 所有功能均通过 Chrome 原生 API 和本地代码实现。

开源与代码

如果您希望验证上述声明，欢迎查看我们的源代码。所有代码均为标准 JavaScript/TypeScript， 不包含任何隐藏的追踪或数据收集逻辑。

政策变更

如果本隐私政策发生变更，我们将通过扩展的更新日志或 Chrome 应用商店的更新说明告知用户。 我们建议您定期查看本页面以了解最新信息。
